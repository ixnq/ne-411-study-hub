# NE-411 Study Hub

Offline interactive revision website for **NE-411**, built from the uploaded lecture notes:

- `Reactivity_Lecture_Notes.docx`
- `Point Kinetics Analytical_lecture Note.docx`

## Open Locally

Open `index.html` in a browser. It works offline as long as the `assets` folder stays beside it.

## GitHub Pages Upload

Upload these files and folders to the root of a GitHub Pages repository:

- `index.html`
- `assets/course-images/reactivity-figure-1.png`
- `assets/course-images/reactivity-figure-2.png`
- `assets/course-images/point-kinetics-figure-1.png`
- `assets/course-images/point-kinetics-figure-2.png`
- `assets/course-images/point-kinetics-figure-3.png`
- `README.md`
- `QUALITY_CHECK.md`

Then enable GitHub Pages for the repository branch. `index.html` is already the entry page.

## Easy Sharing

Use `single-file.html` when you want to share only one file. It embeds the extracted course figures directly in the HTML.

## Features

- Exam cockpit overview with readiness score
- Recommended next topic based on selected study path
- Study path cards with estimated time and progress
- Course breakdown by topic
- Detailed notes with simple-word explanations
- Formula and definition sheet
- Flashcards with local progress
- MCQ quiz with explanations
- Short-answer prompts
- Calculation practice with worked solutions
- One-group point-kinetics calculator
- Final cheat sheet
- Exam-readiness checklist
- Dark mode by default with light mode toggle
- English interface with Arabic UI/simple-summary toggle
- Search and filters
- Local-storage progress tracking
