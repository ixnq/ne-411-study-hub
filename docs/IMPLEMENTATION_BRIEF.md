# Implementation Brief

Refactor the app from a one-course prototype into a generic curated-course platform.

## Next Steps

- Add a course catalog screen.
- Add a course detail screen.
- Define reusable course data.
- Move the existing course into its own course pack.
- Scope progress by course id.
- Keep the existing flashcards, quizzes, formulas, practice, checklist, language toggle, theme toggle, search, and filters working.

## Done Means

- The global app is not branded as one course only.
- The existing course appears as one item in a catalog.
- Adding a new course should require course data, not a full app rewrite.
