# Send This To Classmates

Fastest option:

- Send `NE-411-Study-Hub-Single-File.html`
- They can double-click it and study offline in their browser.

Full folder option:

- Send `NE-411-Study-Hub-Full.zip`
- They should unzip it, then open `index.html`.
- This version keeps the image assets in a normal folder and is best for GitHub Pages.

What is included:

- Overview dashboard with exam-readiness score
- Detailed notes by topic
- Formula and definition sheet
- Flashcards
- MCQ quiz with explanations
- Short-answer practice
- Calculation practice
- Final cheat sheet
- Exam-readiness checklist
- Dark/light mode
- English/Arabic toggle
- Local progress tracking

Note:

- Progress is saved separately on each person’s own browser using local storage.
- No internet is needed after opening the file.
